# Simple Node.js App

A simple Node.js application deployed with Docker and AWS for learning DevOps and cloud deployment.

## Features
- Simple REST API endpoints
- Dockerized for easy deployment
- Deployed on AWS (EC2, ECS, or ECR)
- Demonstrates CI/CD workflow with GitHub Actions (optional)

## Technologies
- Node.js
- Express.js
- Docker
- AWS (ECS, ECR, EC2)
- GitHub Actions (for CI/CD)
